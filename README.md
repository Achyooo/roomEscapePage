## 개요

React 라이브러리를 이용하여 진행한 1인 프로젝트입니다.
<br/>

평소 오프라인 방탈출 카페 컨텐츠를 즐겨 하는 사람으로서, 예약을 위해 필연적으로 방문하게 되었던 방탈출 사이트를 직접 구현해 보고 싶다는 생각을 하여, 해당 프로젝트를 진행하게 되었습니다.
<br/><br/><br/>



## 사용 라이브러리

- react
- redux
- styled-components
- react-router-dom
- react-calendar
<br/><br/><br/>
